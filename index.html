Features

Mobile Friendly

Nuno features a mobile friendly, responsive first website layout using Bootstrap

Full Screen Landing

Using Bootstrap, the Nuno theme features a full screen carousel landing page.

Custom Animation

Animate.css and Waypoints.js allow for smooth animations scrolling down the site.

Parallax Scrolling

Fixed background images allow the theme to have content-filled parallax scrolling sections.

Content Slider

Owl.Carousel.js makes navigating content sliders seamless with its content carousel navigation.

Contact Form

The Bootstrap HTML form will send directly to your email address using PHPMailer.php.
